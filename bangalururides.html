BangaluruRides - Cab Rental Web App

About the Project

BangaluruRides is a full-stack web application I developed to modernize and streamline operations for a local cab rental business. The platform provides a comprehensive booking solution, allowing customers to easily book rides, manage cancellations, and track their cabs in real-time.

The service offers multiple clear and transparent pricing models, including per-kilometer trips, flexible hourly/daily rentals, and all-inclusive tour packages. This project was a personal initiative to support my father's and his associates' cab services, enabling easier customer acquisition and more efficient scheduling.

My Role & Technologies

As the sole developer, I handled the entire development process from concept to deployment. I built the application using the MERN stack and integrated it with Firebase for robust backend services.

  • Frontend: React.js
  • Backend: Node.js, Express.js
  • Database & Auth: Firebase (Firestore, Firebase Auth)
  • Key Features: Fare calculation, order management, user authentication, real-time tracking.
