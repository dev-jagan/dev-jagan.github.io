Beriqa - Emotional Support Platform

About the Project

Beriqa is a commercial platform that provides a safe and confidential space for individuals seeking emotional support. It connects users with certified listeners through chat, audio, or video, ensuring 24/7 availability and privacy.

The platform is built around a moderated community where users can discuss their feelings without judgment. Key features include AI-powered listener matching, mood tracking, and community forums, all designed to provide real conversations and healing.

My Role & Technologies

As a key developer on this project, I was responsible for implementing core features and ensuring a seamless user experience. My work involved both front-end and back-end development to create a robust and scalable platform.

  • Platform: Full-Stack Web Application
  • Core Services: Real-time chat (audio/video), user authentication, AI-powered matching.
  • Key Contributions: Development of the user interface, implementation of the listener matching algorithm, and ensuring the platform's security and confidentiality.
